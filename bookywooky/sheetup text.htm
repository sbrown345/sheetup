A little bookmarklet loader app

Got this idea from Karl Swedberg's jQueryify.

I wanted to extend the idea to allowinga user to load up any library or tools they wanted. For now the tool is limited to Firebug and the libraries are limited to jQuery, Prototype, MooTools and sheetUp. I've quickly tested it on IE6/7, Opera 10, Firefox 3

The default options are jQuery with Firebug but I've added in a few more possible options. If you double click the bookmarklet then it will just load those without showing the diaglog, otherwise the default options will be checked when the dialog opens.
